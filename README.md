# 🌸 Төрсөн өдрийн мэнд – Цэцэг дэлгэрэх анимейшнтэй карт

## Функцууд
- Бодит цэцэг дэлгэрэх анимейшн
- 2 захиа (эхнийх уншсаны дараа хоёр дахь нь үлдэнэ)
- Баяр хүргэх үг
- Бялуу + 22 лаа (10 секунд хүлээх)
- Дөрвөлжин зураг + од харвах
- Дуу тоглуулах (`3.mp3`)
- Өөрийн зураг (`8.jpg`)

## Ажиллуулах
1. `3.mp3` болон `8.jpg` файлуудыг ижил хавтаст хий.
2. Хөтөч дээр `index.html`-г нээ.
3. Эсвэл `python app.py` командаар Flask сервер асаагаад `http://127.0.0.1:5000` руу ор.

## Файлын бүтэц
- `index.html` – бүтэц
- `style.css` – загвар
- `script.js` – логик
- `app.py` – Python сервер
- `3.mp3` – таны дуу
- `8.jpg` – таны зураг
